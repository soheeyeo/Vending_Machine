# 🥤 Vending_Machine
## Overview
HTML/CSS를 활용한 벤딩머신 만들기

## Features
- 미디어 쿼리를 이용하여 모바일 화면(360px)에 맞는 반응형 웹 구현
- root 변수로 반복적인 css 스타일 관리
- 품절 상품 `cursor: pointer` 비활성화
- 상품 리스트에 마우스 hover시 outline 강조
- -webkit-scrollbar 속성으로 좌측 하단 selectedList 스크롤바 스타일 수정
